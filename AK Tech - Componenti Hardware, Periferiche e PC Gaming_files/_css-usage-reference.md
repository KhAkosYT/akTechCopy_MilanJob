# CSS Files Usage Reference - AK Tech

This document provides a reference for CSS files in the AK Tech components directory.

## Files Overview

### app.css
- Primary application styles
- Used for: Main layout and custom components

### awdr_style.css
- WooCommerce Dynamic Pricing styles
- Used for: Price display and discount features

### brands.css
- Brand-specific styling
- Used for: Logo displays and brand-related components

### Core WooCommerce Files

1. woocommerce-layout.css
   - Grid and layout system for shop pages
   - Product lists and detail pages structure

2. woocommerce-smallscreen.css
   - Responsive adjustments for mobile devices
   - Adapts shop layouts for smaller screens

3. woocommerce.css
   - Core WooCommerce styling
   - Product displays, cart, checkout styling

## Vendor-specific Files

1. js_composer.min.css
   - WPBakery Page Builder styles
   - Used for: Page layout components

2. select2.css
   - Enhanced dropdown styling
   - Used in: Forms and selectors

3. style.min.css & style(1).min.css
   - Theme-specific styling
   - Contains: Custom theme overrides and components

## Common Elements and Their Usage

### Product Displays
- `.product` - Individual product container
- `.product-category` - Category display styling
- `.price` - Price display formatting
- `.onsale` - Sale badge styling

### Navigation
- `.main-navigation` - Primary menu styling
- `.nav-menu` - Menu items formatting
- `.dropdown-menu` - Submenu styling

### Layout
- `.site-content` - Main content area
- `.sidebar` - Sidebar styling
- `.widget` - Widget container styling

### Responsive Elements
- Media queries at:
  - 768px (tablet breakpoint)
  - 480px (mobile breakpoint)
  - Custom breakpoints for specific components
